Kréavatar

Avatar generator

View the project

Kréavatar is a fictitious name and avatar generator coded in JavaScript and based on p5.js library.

The program randomly generates a fictitious name, by alternating consonants and vowels to make a name pronounceable and original. Then, a picture is generated, according to different parameters, that can be changed thanks to sliders.
